Projekt címe
Solar System

Projekt leírás
A féléves feladat célja egy interaktív, OpenGL alapú Naprendszer-szimuláció elkészítése C programozási nyelven.
A program a Naprendszer fő elemeinek vizuális megjelenítésére és alapvető mozgásuk szimulálására fókuszál.

A központi elem a Nap, amely nemcsak megjelenítésre kerül, hanem fényforrásként is funkcionál, biztosítva a jelenet megvilágítását.

Megvalósítandó funkciók
Bolygók és gömbök megjelenítése
Bolygók mozgásának szimulálása (keringés, esetleg forgás)
Nap mint fényforrás
Textúrázás a bolygók valósághűbb megjelenítéséhez
Kamera kezelése és mozgatása
(szabad nézet, forgatás, zoom)
Opcionális bővítések

Az alábbi funkciók megvalósítása opcionális, és az idő, illetve a komplexitás függvényében kerülhet be a projektbe:

Gravitáció szimulálása
Amennyiben a fejlesztési idő és a feladat összetettsége lehetővé teszi, a bolygók közötti gravitációs hatások egyszerűsített szimulációja is megvalósításra kerülhet.
Fekete lyuk megjelenítése / szimulációja
További extra funkcióként, ha a projekt időkeretébe belefér és technikailag nem túl bonyolult, egy fekete lyuk vizuális reprezentációja vagy egyszerű hatásmechanizmusa is beépítésre kerülhet.
Felhasznált technológiák
Programozási nyelv: C
Grafikai könyvtár: OpenGL
